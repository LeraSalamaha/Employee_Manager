# Employee Management System

# Описание приложения
Console приложение для управления базой данных сотрудников с полным CRUD функционалом. Приложение разработано в рамках тестового задания и поддерживает работу с SQLite базой данных.


# Основные возможности:
Создание и управление базой данных сотрудников
Добавление, просмотр и поиск записей
Генерация тестовых данных
Оптимизация производительности с индексами
Замер времени выполнения запросов
Поддержка русского и английского языков



# Два режима работы: командная строка и интерактивное меню



# Технологический стек
Язык программирования: Python 3.7+
База данных: SQLite3 (встроенная)
Архитектура: Объектно-ориентированный подход (ООП)
Обработка ошибок: Комплексная валидация и обработка исключений

# Режимы работы
1. Создание таблицы
Создает таблицу employees с полями:

ФИО (full_name)
Дата рождения (birth_date)
Пол (gender)
Временная метка создания (created_at)

2. Добавление сотрудника
Интерактивное или командное добавление записей с валидацией:

Проверка формата даты

Нормализация пола (поддержка русского/английского)

Проверка уникальности (ФИО + дата рождения)

3. Просмотр сотрудников
Вывод всех записей с сортировкой по ФИО и расчетом возраста.

4. Генерация тестовых данных
Создает случайные записей + 100 специальных записей (мужчины с фамилией на "F").

5. Поиск с замером времени
Поиск мужчин с фамилией на "F" с точным замером времени выполнения.

6. Оптимизация базы данных
Создание индексов для ускорения поиска и сравнение производительности до/после оптимизации.
